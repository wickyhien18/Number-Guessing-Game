# Number-Guessing-Game 

https://roadmap.sh/projects/number-guessing-game

This is a simple command-line interface (CLI) game.

## Features

- **Restart:** Start a new game.
- **Hint:** If it's too hard to guess, you can use hints for help.
- **Timer:** Evert time a game start, the score clock begin to run. 
  
  The better you guess with less time the higher the score. .
- **Score:** If you guess the correct number, higher score will be saved and current score will be displayed.
# Installation

1. **Clone the repository:**

```bash
git clone https://github.com/wickyhien18/Number-Guessing-Game
cd Number-Guessing-Game
```

2. **Compile the source code:**

```bash
javac Game.java GameProgram.java
```

3. **Start game:**

```bash
java GameProgram.java
```
